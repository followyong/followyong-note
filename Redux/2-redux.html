第二节：引入Redux

Redux 是数据流管理工具，使用Redux的最重要的一句：

一切数据都要保存到Store之中，组件自己不保存自己的state数据

创建 Store

在 src文件夹下创建store.js，把state的数据存到store.js之中。

装包 redux

npm i --save redux

在store.js中引入{createStore}

import {createStore} from 'redux'

导出store.js

export default store

rudex.js.org 官网 http://redux.js.org/docs/api/createStore.html createStore官网

在src下创建reducers/comment.js，里面的内容为：

export default function commentReducer(state = [] ,action){
  return state
}

在store.js里导入comment.js

从store中读取数据

在要操作的组件中输出 console.log(store.getstate())

如何修改store里的数据

store.dispatch({type:'ADD_COMMENT',comment:newComment})

要操作的组件发出命令，reducer接收

在reducer 里添加

switch(action.type){
  case 'ADD_COMMENT':
    return [...state,action.comment]
  default:
    return state  //case匹配不上就执行default里的内容  
}

createStore接收三个参数

createStore(reducer, [preloadedState], [enhancer])
