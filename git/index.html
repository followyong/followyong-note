项目上线部署

项目上线部署

在github上创建一个新的仓库 将自己的本地项目初始化：git init

git add .

git commit -m 'first commit'

git remote add origin git@github.com:followyong/followyong-note.git

git push -u origin master

npm run build

将打包好的文件剪切到本地仓库外，创建并切换分支gh-pages

git checkout -b gh-pages

把仓库里的文件删除的只剩下node_modules和.gitignore文件

git push -u origin gh-pages

打开网址：https://followyong.github.io/followyong-note 就可以看到自己上线的项目了
